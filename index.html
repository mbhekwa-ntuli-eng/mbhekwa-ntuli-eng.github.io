Featured Control Projects

Three projects showing the progression from modelling to controller design and implementation.

Aircraft Suspension System Identification

Modelled an aircraft suspension system from time-domain and frequency-domain tests. Estimated damping ratio, natural frequency, and transfer function parameters, then validated the model using step and Bode response comparisons.

System Identification Bode Plot MATLAB Second-Order Systems

Proportional Control of a Vertical Lift System

Designed a proportional controller for the kitticopter altitude system. Identified plant dynamics, selected gain from overshoot and robustness specifications, simulated the design, and built the analog controller on veroboard.

Proportional Control Simulink Op-Amp Circuit Veroboard
View PDF

Lead Compensator Design and Implementation

Improved the kitticopter response using a lead compensator designed from root-locus methods. Validated the compensated system in MATLAB/Simulink and implemented the circuit in hardware to improve settling time and reduce overshoot.
